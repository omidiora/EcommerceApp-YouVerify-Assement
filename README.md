
## Running the iOS App (FinanceApp)

Follow these steps to run the FinanceApp on your local machine using Xcode.

### 1. Clone the Repository
```bash
https://github.com/omidiora/EcommerceApp-YouVerify-Assement/new/main?filename=README.md
cd EcommerceApp-YouVerify-Assement
```

## Open the Xcode Project

If you see a EcommerceApp.xcworkspace file (CocoaPods), open it:

open ios/EcommerceApp-YouVerify-Assement.xcworkspace


 Otherwise, open the .xcodeproj file:

open ios/EcommerceApp-YouVerify-Assement.xcodeproj

 ## Install Dependencies

If the project uses CocoaPods:

cd ios
pod install
cd ..
